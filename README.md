# Student-Achievement-Analysis-Dashboard---Data-Modelling-
An interactive Microsoft Excel dashboard for analyzing student achievement data through KPI cards, Pivot Tables, Pivot Charts, and dynamic slicers. It visualizes awards, accumulated points, achievement categories, regions, study programs, and competition levels to support efficient data analysis and reporting.

## ✨ Features

- 📌 KPI Cards
  - Total Awards
  - Total Accumulated Points
  - Average Points per Achievement

- 📊 Interactive Charts
  - Bar Chart showing total achievement points by students' region of origin.
  - Pie Chart illustrating the distribution of achievement categories.

- 🎛 Dynamic Slicers
  - Region of Origin
  - Study Program
  - Competition Level

- 🔄 Automatic Dashboard Updates
  - All KPIs and charts update dynamically based on selected filters.

---

## 📈 Dashboard Overview

This dashboard allows users to:

- Monitor overall student achievement performance.
- Compare accumulated achievement points across different regions.
- Analyze the distribution of achievements by category.
- Filter data based on study program and competition level.
- Generate meaningful insights through interactive data visualization.
